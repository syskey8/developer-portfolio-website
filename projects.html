TRAVEL MANAGEMENT SYSTEM

I made this application as a college project. This application has java as its front end and MySql has been used for the backend purpose to store data like accounts, credentials etc.

Swing API is been used to make a window based output. Swing classes like Jbutton, JProgessBar, ActionListener etc. We have used MySQL for backend as our database, where we store data related to accounts, packages, customer details etc. Our MySQL database design for the Travel Management System is a cornerstone of this project. During Loging in it checks up with database to see the combination of username or password is correct, if not then it give invalid login credentials dialogue box, it also retrieves data regarding customer details and booked packages details.

VIRTUAL MOUSE

I made this application as a college project. This application is made solely on python. This application uses open source APIs from google like mediapipe, OpenCV, numpy etc.

In this HandTracking module's working is demonstrated, it plots 20 points across the hand and those are recorded for further use

In this the Module only considering primary hand is shown, lastest hand in the feed is considered as primary hand.

In this the pointer mode is turned on, the cue for the same is the index finger. The mouse will move as we move the index finger

In this the click mode is turned on, as the middle finger and index fingers are visbile in the feed.

Click mode constantly calculates the distance between the two fingers if it is less than 50px it triggers click.
